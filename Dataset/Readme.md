# Dataset

This project uses the APTOS 2019 Blindness Detection dataset.

Download it from:
[https://www.kaggle.com/competitions/aptos2019-blindness-detection
](https://www.kaggle.com/datasets/mariaherrerot/aptos2019)
After downloading, place the dataset in the following structure:

dataset/


├── train_images/


├── test_images/

└── train.csv
